# 🐟 수산물 건강사전 (ver 3.4)

효능별(7개) + 수산물별(60개) + 세부정보(6개) 기반의  
**인터랙티브 수산물 건강 데이터 사전**입니다.


## ✅ 주요 기능
- **효능 선택** (고혈압 예방 / 간 기능 향상 / 성인병 예방 / 스태미나 / 피부미용 / 성장발육 / 기타)
- **수산물 60종** 선택 또는 검색 입력하여 바로 조회
- **세부정보 제공** (영양 정보 / 조리법 / 유통 정보 등)
- 검색창에서 효능 또는 수산물을 입력 → 자동 매칭 결과 표시
